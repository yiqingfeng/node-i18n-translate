## node-i18n-translate

多语言自动翻译，用于国际化处理

### 使用

快速使用：

```
npx github:yiqingfeng/node-i18n-translate -V
```

> **注意：**
>
> 使用 npx 形式时，文件路径
>
> 不要使用相对路径，英文此时的相对路径是相对 npx 所在位置的

常规使用：

```
git clone https://github.com/yiqingfeng/node-i18n-translate.git
cd node-i18n-translate
npm install
npm start
```
